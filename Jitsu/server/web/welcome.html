Welcome to Jitsu!

If you see this page, the Jitsu server is successfully installed and working.

Add following config parameter to disable this page (blank page will be shown instead)

server:
  disable_welcome_page: true

This server is configured by Jitsu configurator. Please, visit Jitsu Configurator page to add or remove destinations, connectors or API keys.

Read more about event tracking at JS SDK documentation section.
